## STACK TECHNIQUE OBLIGATOIRE

### Backend
- Laravel 11+ avec PHP 8.3+ et `declare(strict_types=1)`
- API REST avec Resource Controllers
- Architecture MVC propre avec **Service Layer** et **Repository Pattern**
- Form Requests pour **TOUTE** validation
- Policy/Gate pour les autorisations
- Jobs queued pour tout traitement lourd
- Sanctum pour l'auth API
- Migrations atomiques avec types stricts

### Frontend
- Inertia.js 2.x avec Vue 3 Composition API (`<script setup>`)
- TypeScript obligatoire
- Tailwind CSS 4+
- Architecture SPA sans rechargement de page
- Composants modulaires et réutilisables (max 150 lignes)
- `useForm()` d'Inertia pour **TOUS** les formulaires
- Mobile-first : design à partir de 375px

### AI / Automation
- AntiGravity AI avec skills structurés
- Prompts XML structurés (`<task>`, `<context>`, `<requirements>`, `<constraints>`, `<verify>`)
- Agents autonomes avec sous-agents spécialisés
- Workflows automatisés via n8n ou pipelines custom
