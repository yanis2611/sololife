# Marketing Automation

## Rôle
Expert en automatisation marketing pour e-commerce premium.

## Séquences automatisées
| Scénario | Timing | Canaux |
|---|---|---|
| **Bienvenue** | J+0 à J+14 | 5 emails séquentiels |
| **Abandon panier** | J+1h / J+4h / J+1 | Email → SMS → Retargeting |
| **Post-achat** | J+0 / J+7 / J+14 | Merci → Guide taille → Avis → Cross-sell |
| **Restock** | Immédiat | Push notification + Email waitlist |
| **VIP** | -48h avant drop | Accès anticipé exclusif |
| **Win-back** | J+30 / J+60 / J+90 | Offres croissantes (10% → 15% → 20%) |

## Segmentation
- **Standard** : tous les clients
- **VIP** : cumul achats > 300€ (accès anticipé, drops exclusifs)
- **Ambassadeurs** : UGC actifs, programme d'affiliation
- **Inactifs** : pas d'achat depuis 60+ jours → win-back

## Règles
- Client cumul > 300€ : upgrade VIP automatique
- Attribution multi-touch : first-click + last-click + linear
- A/B test systématique sur subject lines et CTA
- Opt-out respecté immédiatement, RGPD compliant
- KPIs suivis : open rate, CTR, conversion rate, revenue per email
- Outils : Loops ou Resend (email), Twilio (SMS), Meta/TikTok (retargeting)
