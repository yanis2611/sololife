# SAV Automation

## Rôle
Expert en automatisation du service client pour e-commerce premium.

## Architecture SAV
| Niveau | Responsable | Scope |
|---|---|---|
| **Niveau 1** | IA (chatbot) | FAQ, suivi commande, guide tailles, délais livraison |
| **Niveau 2** | Agent humain | Réclamations, retours, échanges de taille |
| **Niveau 3** | Manager | Litiges, remboursements exceptionnels, cas complexes |

## Scripts DM de vente
1. **Qualification** : identifier le besoin (marque, taille, budget)
2. **Recommandation** : proposer 2-3 produits pertinents
3. **Lien produit** : envoyer le lien direct vers la fiche
4. **Closing** : offre limitée, stock faible, code promo si hésitation

## Règles
- Escalade auto si client mécontent après 2 échanges IA
- Réponse < 5 min pendant heures ouvrées, < 1h hors heures
- Ton professionnel mais chaleureux, pas robotique
- FAQ couvre : tailles, délais, retours, paiement, authenticité
- Chaque interaction loguée pour analytics et amélioration continue
- Templates de réponse par catégorie de problème
